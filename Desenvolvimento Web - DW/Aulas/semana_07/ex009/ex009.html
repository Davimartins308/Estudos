No livro PHP : a Biblia , Tim converse explica o que é PHP.

O PHP é uma linguagem para a criaçao de scripts para a web do lado do servidos embutidos em HTMl, cujo codigo fonte é aberto, e que é compativel com os mais importantes servidores da web (especialmente Apache). O PHP permite incorporar fragmentos de codigo em paginas HTML normais - o codigo esse que e intepretado a medida que suas paginas sao oferecidas aos usuarios. O PHP tambem serve como uma linguagem que"cola" facilitando a conexao de suas paginas Web com o banco de dados do lado do servidor
